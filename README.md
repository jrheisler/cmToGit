# cm_git

A new Flutter project.

## Getting Started

This project is a starting point for a Flutter Web application.

Goals:
1. Create a system to allow students to create change packages
  a. CRUD Change Packages
2. Use Git
  a. Attach change packages to files that change in a Git repo
  b. Track the change packages through Git.
2. Show the results in a Kanban
  a. Create an interactive Kanban
  b. With a variable number of columns
  c. Using drag and drop
  d. Column level limits
3. Lists of Users
  a. Full CRUD   
5. Local Data
  a. Settings for the user to tie to a Git Repo
6. Backlog
7. Finished
